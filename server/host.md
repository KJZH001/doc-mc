# 🏯服务器地址
---

java版地址 支持1.13-1.20.4（服务器列表可能错误显示为不支持，无实际影响）

基岩端需要1.20.60/1.20.80，其余版本不受支持


## java版
* 优先推荐（需要ipv6），一般情况下速度更快，更稳定
`mcv6.moeworld.top`
`mc.v6.moeworld.top:35578`
* 备用地址（ipv4支持）
`mcpc.moeworld.top`
`mc.moeworld.top`
`mc.moeworld.top:35578`

## bedrock基岩版

仅ipv4和外置登录

`mcbe.moeworld.top:19132`
