# 🏯 服务器地址

---

java 版地址 支持 1.13-1.20.4（服务器列表可能错误显示为不支持，无实际影响）

基岩端需要 1.20.60/1.20.80，其余版本不受支持

## java 版

- 优先推荐（需要 ipv6），一般情况下速度更快，更稳定

  `mcv6.moeworld.top`

  `mc.v6.moeworld.top:35578`

- 备用地址（ipv4 支持）

  `mcpc.moeworld.top`

  `mc.moeworld.top`

  `mc.moeworld.top:35578`

## bedrock 基岩版

仅 ipv4 和外置登录

- 推荐：

  `mcbe.moeworld.top:19132`

- 备用：

  `106.75.247.209:19132`
